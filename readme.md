# 环境
1. 创建launch.json
   按debug按钮，按照提示创建launch.json
   // 2020/10/20新版本的vscode，只要配置好g++的path，点一下那个创建launch.json之后，
2. 创建tasks.json （配置g++ 参数）
   方法1 源码文件上按F5,按照提示创建tasks.json
   方法2 shift + ctrl + p -> Tasks:Configure Task -> C/C++ g++.exe build active file
